# Parcel-Management-System

This is a parcel management system built using Java. 

<h2>Technologies:</h2>

* Eclipse Java EE IDE
* Apache Derby
* Tomcat Server
* Advanced Java Concepts (JSP, Servlet, JDBC)
* UI (HTML, CSS, JS)
* SQL
